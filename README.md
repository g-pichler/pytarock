# PyTarock

PyTarock ist ein Online Tarock bzw. [Königrufen](https://de.wikipedia.org/wiki/K%C3%B6nigrufen).
Es besteht aus 2 Teilen:
* JavaScript Front-End in `htdocs/`
* Python WebSocket Backend `TarockWSServer.py`
